GameStore

Esta aplicación tiene como función principal la gestión de la base de datos de una tienda de videojuegos con la ayuda de hibernate.

Hibernate es una herramienta de mapeo objeto-relacinal(ORM) para la plataforma Java, que facilita el mapeo de atributos entre una base de datos relacional tradicional y el modelo de objetos de una aplicación, mediante archivos declarativos(XML) o anotaciones en los beans de las entidades que permiten establecer estas relaciones.

Hibernate es software libre, distribuido bajo los términos de la licencia GNU LGPL.
